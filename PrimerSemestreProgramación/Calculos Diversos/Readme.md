# Practise for Programming I Finals.
A directory that has some exercises regarding core concepts from Python.

## Current content:

### calculos_lluvias.py :
- An algorithm that with an array of a quantity of rain it can calculate some operations.
- These operations can be:  a streak of days that didn't rain, the number of max quantity of rain or the min.

### calculos_temperaturas.py
- Similar with **calculos_lluvias.py** but with different application to temperature.
- With operations like: average temperatura, max, min, streak of negative temperature, or change an exact index.
